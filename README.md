# Ejercicio 1: Clasificar Matrices

Identifica el tipo de cada matriz:

*Matriz A:*

| 1 | 0 |
| 0 | 1 |

*Matriz B:*

| 3 | 0 | 0 |
| 0 | -2 | 0 |
| 0 | 0 | 5 |

*Matriz C:*

| 2 | 1 | 4 |
| 1 | 3 | 5 |
| 4 | 5 | 6 |

*Matriz D:*

| 1 | 2 | 3 |
| 0 | 4 | 5 |
| 0 | 0 | 6 |

## Respuesta

- *A:* Matriz Identidad 
- *B:* Matriz Diagonal
- *C:* Matriz Simétrica
- *D:* Matriz Triangular Superior

# Ejercicio 3: Multiplicación en Cadena

## Enunciado

Verificar que (AB)C = A(BC) con las matrices:

*Matriz A:*

| 1 | 2 |
| 3 | 4 |

*Matriz B:*

| 2 | 0 |
| 1 | 3 |

*Matriz C:*

| 1 | 1 |
| 0 | 2 |

## Solución

*AB =*

| 4 | 6 |
| 10 | 12 |

*(AB)C =*

| 4 | 16 |
| 10 | 34 |

*BC =*

| 2 | 2 |
| 1 | 7 |

*A(BC) =*

| 4 | 16 |
| 10 | 34 |

*Conclusión:* (AB)C = A(BC) ✓
